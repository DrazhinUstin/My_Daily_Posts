# My Daily Posts

This project was created using [Vite](https://vitejs.dev) and [Firebase](https://firebase.google.com).

## Installation

1. Download zip file or Clone Repo
2. Open your Terminal and `cd` to your local copy of the repo
3. Run `npm install`
4. Run `npm run dev`
